# Fitnation Backend

## Requirements

You need to have 
- docker
- docker-compose
- ruby-2.7.2
- rails
- bundle

*`docker-compose` is used for `mysql` engine*
## Run instructions

1. Run `sudo docker-compose up -d` to start the database engine, or run your mysql server yourself
2. Run `bundle install`
3. Run `rails server`
